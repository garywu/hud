## THE STORY

Jane is not a chatbot. She's an always-present AI companion living in your MacBook notch.

**Before:** Open ChatGPT tab → type → close → she forgets
**After:** "Jane, what did Sarah recommend?" → instant voice answer → she remembers from last week

Jane has:
- **A face** — expressive avatar in the notch
- **A voice** — hears you through wake word, speaks with natural TTS
- **Memory** — remembers conversations spanning weeks/months
- **Local superpowers** — accesses files, controls apps, monitors system
- **Open source DNA** — MIT licensed, no vendor lock-in, self-hosted option

**Market Gap:** No product combines persistent memory + voice interaction + local machine access + ambient visual presence + deep AI reasoning. Jane fills this gap.
